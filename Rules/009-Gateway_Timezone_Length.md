# Rule 9: Gateway Timezones Length

```
http://docs.aws.amazon.com/storagegateway/latest/APIReference/API_ActivateGateway.html#API_ActivateGateway_RequestSyntax
```
A value that indicates the time zone you want to set for the gateway. 
The time zone is used, for example, for scheduling snapshots and your 
gateway's maintenance schedule.

This is also represented in a String. This String has a min and a max

For AWS: min 3. max 10.
